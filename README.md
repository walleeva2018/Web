# Step 1
Download the project as zip file from upper right corner 

# Step 2
Open the folder in terminal

# step 3 
Move to frontend folder 
```
 cd frontend
```
# step 4 
Download the node modules
```
npm install
```
# step 5
Build the scripts
```
npm run-script build
```
# step 6
Move to back end 
```
cd ..
cd backend
```
